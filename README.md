# vue-nonet

提供一种解决方案，解决大数据量情况的数据渲染性能问题，是基于多维度多层级的九宫格算法。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
